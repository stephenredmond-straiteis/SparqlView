SparqlView is a visual RDF graph explorer that lets you browse linked data resources via SPARQL endpoints with a simple and friendly interface.

DOWNLOAD SparqlView

Resources published according to the W3C SPARQL standard are easy to reach and understand. You can start browsing by querying the endpoint for a specific resource, or start from one of the provided URI examples.

SparqlView includes a built-in SPARQL endpoint backed by local Turtle files, a browser-based query editor, and this visual graph navigator.

SparqlView is a visual RDF graph explorer, based on an experimental legacy project called 'LodLive' created by Diego Valerio Camarda, Silvia Mazzini, and Alessandro Antonuccio that has been updated by Stephen Redmond of Straitéis AI.

SparqlView allows you to navigate RDF resources via SPARQL endpoints, connecting resources across multiple endpoints by making use of LOD interconnection capacities such as owl:sameAs.

It includes a built-in read-only SPARQL endpoint backed by local Turtle files, a browser-based query editor, and this visual graph navigator for exploring linked data with a simple and friendly interface.

Here you can follow the direct and inverse connections provided by the resource by moving freely from one "circle" to the others.

Every new pop-open resource will guide the user to those relating to it, automatically connecting the new "circle" to the ones they have already opened.

Every time an owl:sameAs property is reached throughout the course of navigation, SparqlView will connect to the related endpoint, enabling the user to move to the new available resources within the same navigation environment.
